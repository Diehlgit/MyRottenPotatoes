Nesses arquivos definimos e criamos o Banco de Dados do nosso programa.

Em 'migrate' criamos uma migração do esquema do Banco de Dados, depois 
usamos o comando 'rails db:migrate' para efetuar essa migração.

Em 'schema.rb' temos o esquema criado pela migrate.

Em 'seeds.rb' populamos nosso Banco de Dados com alguns filmes.
